## Project Context & AI Guidelines

**Target Audience:** Vietnamese tourists and travelers visiting Vietnam.
**Language:** All UI text must be in **Vietnamese** (Tiếng Việt).
**Currency:** All prices must be displayed in **VNĐ** (Vietnamese Dong) and formatted accordingly (e.g., 1.500.000đ).
**Content Focus:** All mock data and placeholders must relate to **Vietnam locations** (e.g., Ha Long Bay, Da Nang, Phu Quoc) instead of international examples.
